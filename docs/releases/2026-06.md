# 2026-06 发布记录

## v2026-06-19

### 新功能
- 问题评论实时聊天:右下角聊天气泡(微信/Intercom 风格),问题有新评论时负责人、协助人及评论中被 `@` 的人实时收到提醒,可在气泡内直接回复(= 新增评论);每个问题 = 一个会话。(#14)
- 每个项目支持多数据库备份引擎:按项目配置多个数据库备份目标。(#13)
- Issue 状态支持禁用;状态/优先级颜色选项 widget 支持原始 JSON 编辑。(#12)
- 个人资料支持自定义头像上传与裁剪,新增「卡通插画」头像组。(#14)

### 改进
- 登录页支持显示密码;评论支持 Cmd/Ctrl+Enter 快捷发送。(#14)
- Markdown 图片支持 `|w=` 宽度标记(高度自适应)与 `|left`/`|center`/`|right` 对齐标记。(#14)

### 缺陷修复
- 修复 Markdown 预览组件初始化时序问题(TDZ),避免渲染报错。(#14)

## v2026-06-16-2

### 新功能
- PR ↔ Issue 关联:自动解析 PR 文本中的 ISS-xxx 引用并回填关联;Issue 详情页展示关联 PR,PR 合并后建议将 Issue 标记为「已解决」;仓库详情页新增 Pull Requests 页签;手动同步与每小时任务一并拉取 PR。
- Issue 列表新增「提出人」筛选下拉与「只看我提出的」快捷按钮(按列表显示的提出人匹配:reporter 文本,为空则回退到创建人)。
- 负责人候选限定为「开发者」用户组成员(列表筛选、新建、批量分配、详情页、分配/转交弹窗)。

### 改进
- PR 状态徽标中文化,同步按钮文案优化。
- PullRequest 新增后台管理入口。

### 缺陷修复
- Issue 详情页隐藏结算金额展示。
- 钳制超长 ISS 编号,避免 PR 同步因数字过大而崩溃。
